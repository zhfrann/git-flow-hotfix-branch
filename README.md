# MyApp

A simple task management web application built for demonstration purposes.

## Overview

MyApp helps teams track tasks, assign responsibilities, and monitor project progress in real time. This repository follows the **Git Flow** branching strategy, including hotfix branches for urgent production fixes.

## Features

- Create, update, and delete tasks
- Assign tasks to team members
- Set priority levels and due dates
- Real-time status tracking (To Do → In Progress → Done)
- Email notifications on task updates

## Branching Strategy

| Branch | Purpose |
|---|---|
| `main` | Production-ready code |
| `develop` | Integration branch for features |
| `feature/*` | New feature development |
| `release/*` | Release preparation |
| `hotfix/*` | Urgent production bug fixes |
